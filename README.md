# Configuration-management

## Этап 1: REPL
- Реализован графический интерфейс (GUI)
- Заголовок окна формируется на основе данных ОС
- Парсер с поддержкой раскрытия переменных окружения ($HOME)
- Команды-заглушки: ls, cd
- Команда exit для выхода
- Обработка ошибок выполнения команд

## Этап 2: Конфигурация
### Реализованные функции:
- **Параметры командной строки**:
  - `--vfs PATH` - Путь к файлу VFS
  - `--log PATH` - Путь к лог-файлу  
  - `--script PATH` - Путь к стартовому скрипту
  - `--debug` - Включить отладочный вывод
- **Логирование** - События вызова команд записываются в CSV файл с полями:
  - Дата и время события
  - Выполненная команда
  - Статус выполнения (успех/ошибка)
  - Сообщение об ошибке (если есть)
- **Стартовый скрипт** - Поддерживает комментарии (строки начинающиеся с #)
- **Отладочный вывод** - При запуске с `--debug` выводятся все заданные параметры
- **Обработка ошибок** - Сообщает об ошибках во время исполнения стартового скрипта

## Этап 3: VFS

### Реализованные функции:
- **Виртуальная файловая система** - Все операции производятся только в памяти
- **Загрузка из XML** - VFS загружается из XML файла со структурой:
  - `<directory name="...">` для каталогов
  - `<file name="..." type="file">` для файлов
- **Поддержка base64** - Двоичные данные кодируются в base64
- **Обработка ошибок** - Сообщает об ошибках загрузки VFS (файл не найден, неверный формат)
- **Многоуровневая структура** - Поддержка не менее 3 уровней вложенности файлов и папок
- **MOTD** - Файл `motd` в корне VFS выводится при старте эмулятора

### Тестирование этапа 3:

#### Тест 1: Загрузка VFS из XML
```bash
python ShellEmulator.py --vfs VFS.xml --debug

## Этап 4: Основные команды

### Реализованные функции:
- **Команда ls** - Полная реализация с поддержкой VFS:
  - Вывод содержимого текущего каталога
  - Поддержка указания пути: `ls /home/user`
  - Обработка ошибок: несуществующие пути, файлы вместо каталогов
- **Команда cd** - Полная реализация с поддержкой VFS:
  - Смена текущего каталога
  - Поддержка относительных и абсолютных путей
  - Специальные пути: `.`, `..`, `/`
  - Обработка ошибок: несуществующие пути, файлы вместо каталогов
- **Команда echo** - Вывод текста с поддержкой аргументов
- **Команда cal** - Вывод календаря:
  - Без аргументов - текущий месяц
  - С одним аргументом - указанный месяц текущего года
  - С двумя аргументами - указанные месяц и год
  - Обработка ошибок: неверные номера месяцев
