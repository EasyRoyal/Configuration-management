# Configuration-management

## Этап 1: REPL
- Реализован графический интерфейс (GUI)
- Заголовок окна формируется на основе данных ОС
- Парсер с поддержкой раскрытия переменных окружения ($HOME)
- Команды-заглушки: ls, cd
- Команда exit для выхода
- Обработка ошибок выполнения команд

## Этап 2: Конфигурация
### Реализованные функции:
- **Параметры командной строки**:
  - `--vfs PATH` - Путь к файлу VFS
  - `--log PATH` - Путь к лог-файлу  
  - `--script PATH` - Путь к стартовому скрипту
  - `--debug` - Включить отладочный вывод
- **Логирование** - События вызова команд записываются в CSV файл с полями:
  - Дата и время события
  - Выполненная команда
  - Статус выполнения (успех/ошибка)
  - Сообщение об ошибке (если есть)
- **Стартовый скрипт** - Поддерживает комментарии (строки начинающиеся с #)
- **Отладочный вывод** - При запуске с `--debug` выводятся все заданные параметры
- **Обработка ошибок** - Сообщает об ошибках во время исполнения стартового скрипта
